What is jQuery?

jQuery is a cross-browser, cross-platform Javascript library designed to simplify client-side Javascript scripting.[1]

jQuery allows you to change, append to, remove, and perform other operations on DOM elements.

[1]: https://en.wikipedia.org/wiki/JQuery

Different parts of jQuery

jQuery: The core library that allows you to do Javascript functions more easily.

jQuery UI: The addition to jQuery that provides more functionality for cool user interface stuff like drag and drop, animation, and element interactivity.

jQuery Mobile: The addition to jQuery that optimizes jQuery and jQuery UI performance on mobile devices.

Including jQuery in your website

Add the following lines of markup to your HTML files to include jQuery, preferably in the <head> section:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.0/jquery.min.js"></script>
<script src="https://ajax.googleapis.com/ajax/libs/jqueryui/1.11.4/jquery-ui.min.js"></script>
<script src="https://ajax.googleapis.com/ajax/libs/jquerymobile/1.4.5/jquery.mobile.min.js"></script>

The numbers are the versions of each module; update those as updates come out for them.

Example jQuery code

$(document).ready(function(){
$("a").click(function(event){
alert("You can't click this link!");
event.preventDefault();
});
});
